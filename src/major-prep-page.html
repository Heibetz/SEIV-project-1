How to Prepare to be a CS Major

Use these sections to focus on the key skills, courses, and habits that will help you succeed as a CS major.

Core Skills
  • Strong problem solving and analytical thinking
  • Basic programming knowledge (variables, loops, functions)
  • Comfort with debugging and testing code
  • Math foundation: logic, sets, and basic proofs
Starter Courses / Topics
  • Programming 1 & 2
  • Object-Oriented Programming
  • Data Structures & Algorithms
  • Intro to Discrete Mathematics
Recommended Tools

Soft Skills
  • Clear communication (explain your code)
  • Teamwork and collaboration
  • Time management and consistency
  • Adaptability and curiosity
