To prepare for a major in Computer Science, focus on problem-solving, programming practice, and math fundamentals. Consistency matters more than perfection.

Core Skills

  • Programming basics: variables, conditionals, loops, functions
  • Debugging: read errors, print/log, use a debugger
  • Discrete math: logic, sets, functions, basic proofs
  • Git basics: add → commit → push; write a clear README

Starter Courses / Topics

  • Intro to Programming (Python or Java)
  • Data Structures (lists, stacks, queues, maps)
  • Discrete Mathematics
  • Optional: Web basics (HTML/CSS/JavaScript) or Databases (SQL)

Simple Weekly Actions

  • Build or improve a tiny project (1–2 hours)
  • Push code to GitHub once per week
  • Do 5–10 practice problems (HackerRank/LeetCode easy)
  • Attend office hours / tutoring or a CS club meetup
